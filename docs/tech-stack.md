# Prigenix Tech Stack

We are technology-agnostic in principle and opinionated in practice. We choose tools based on the problem, the existing client environment, and long-term maintainability.

## Frontend

- **React** — Component-based UI development.
- **Next.js** — Server-side rendering, static generation, and API routes.
- **TypeScript** — Type safety across the codebase.
- **Tailwind CSS** — Utility-first styling for fast, consistent UI.
- **Framer Motion** — Rich animations and transitions.

## Backend

- **Node.js** — High-performance APIs and microservices.
- **Python** — AI/ML services, data pipelines, and automation scripts.
- **Go** — High-throughput services and command-line tools.
- **PostgreSQL** — Primary relational database.
- **MongoDB** — Document-oriented data when schema flexibility is needed.
- **Redis** — Caching, queues, and real-time state.

## AI & Machine Learning

- **OpenAI / Anthropic APIs** — LLM-powered features and agents.
- **LangChain / custom orchestration** — Agent workflows and chaining.
- **Vector databases** — Pinecone, Weaviate, pgvector for RAG and semantic search.
- **Hugging Face** — Open-source models and embeddings.

## Cloud & Infrastructure

- **AWS** — Primary cloud provider (EC2, Lambda, S3, RDS, ECS, EKS, IoT Core).
- **Azure & GCP** — Used when clients already operate there.
- **Docker & Kubernetes** — Containerization and orchestration.
- **Terraform & Pulumi** — Infrastructure as code.
- **GitHub Actions / GitLab CI** — CI/CD pipelines.

## Industrial IoT

- **MQTT & OPC-UA** — Device messaging and protocol translation.
- **Modbus & BACnet** — Legacy device integration.
- **InfluxDB & TimescaleDB** — Time-series data.
- **Grafana** — Operational dashboards.
- **AWS IoT / Azure IoT Hub** — Cloud IoT platforms.

## Security & Compliance

- OAuth 2.0, OpenID Connect, SAML
- Encryption at rest and in transit
- Role-based access control (RBAC)
- SOC 2, GDPR, HIPAA-aligned practices when required
